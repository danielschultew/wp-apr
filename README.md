# wp-apr
This is a fork of the original plugin from https://profiles.wordpress.org/nerdywp/

A WordPress Plugin for Auto Product Restock every Week.

You can select the products, the amount, the day and time the products should be restocked.

Installation
_____________
1. Upload and install the .zip file through the 'Plugins' menu in WordPress
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Start using the plugin
